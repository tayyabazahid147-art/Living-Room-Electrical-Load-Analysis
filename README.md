# Living-Room-Electrical-Load-Analysis
Living Room Electrical Load Analysis

Electrical Engineering Mini Project

Overview
This project analyzes the electrical load of a residential living room using circuit analysis and Proteus simulation.

Objectives
• Calculate current using Ohm's Law
• Determine load resistance
• Calculate monthly energy consumption
• Estimate electricity cost
• Verify results through Proteus simulation

Software
• Proteus 8 Professional
• Microsoft Excel

Theory
• Ohm's Law
• Power Equation
• Parallel Circuits
• Energy Consumption

Results

✔ Calculated currents verified by simulation

✔ Monthly energy consumption estimated

✔ Electricity bill calculated
